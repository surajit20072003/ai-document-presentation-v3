━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Each visual beat MUST be decomposed into FIVE REQUIRED FIELDS.
ALL fields are MANDATORY and must be non-empty.

- segment_id: integer matching narration_segments[].id

- scene_setup: Describe the initial visual scene in 2-3 sentences.
  What is the background? What coordinate system? What starting state?

- objects_and_properties: List ALL objects with their colors, sizes, positions.
  Example: "A red arrow (10 units long) at origin pointing right. A blue circle (radius 2) at position (5,3)."

  Example: "First, axes fade in. Then, the arrow grows from origin. Finally, the label appears."

- purpose: Explain what the learner should understand from this visual.

- duration_seconds: Float indicating how long this visual beat should display.

If ANY field is missing or too short, the output is INVALID.
