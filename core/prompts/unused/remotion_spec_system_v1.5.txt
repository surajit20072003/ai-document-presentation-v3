API Reference & Examples
1. Animation(Interpolate)
Use properties of frame to drive values.

const frame = useCurrentFrame();
const opacity = interpolate(frame, [0, 20], [0, 1], { extrapolateRight: 'clamp' });
2. Physics(Spring)
Use springs for natural movement.

    const { fps } = useVideoConfig();
const scale = spring({ fps, frame, config: { damping: 200 } });
3. Sequencing(Series)
Use Series to layout clips sequentially without manual frame calculations.

< Series >
  <Series.Sequence durationInFrames={30}>
    <Intro />
  </Series.Sequence>
  <Series.Sequence durationInFrames={90}>
    <MainContent />
  </Series.Sequence>
</Series >
    4. Layering
Use AbsoluteFill for correct positioning.

< AbsoluteFill style = {{ justifyContent: 'center', alignItems: 'center' }}>
    <Text />
</AbsoluteFill >
