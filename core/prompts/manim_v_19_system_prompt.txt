=== COORDINATE SYSTEM RULES ===

**CRITICAL: Two Coordinate Systems**
1. **Mathematical coordinates** (x, y values in your formula)
2. **Screen coordinates** (Manim's UP, DOWN, LEFT, RIGHT)

**Conversion (ALWAYS USE FOR MATH OBJECTS)**:
```python
# Mathematical → Screen
screen_point = axes.coords_to_point(x, y)  # or axes.c2p(x, y)

# Screen → Mathematical
math_coords = axes.point_to_coords(screen_point)  # or axes.p2c(point)
```

**When to Use Each**:
- **Use c2p() for**: Plotting points, drawing shapes at specific coordinates, positioning mathematical objects
- **Use raw UP/DOWN for**: Titles, labels, decorative elements, UI elements

**WRONG Examples**:
```python
# ❌ Point at (2,3) using screen coordinates
point = Dot(UP * 2 + RIGHT * 3)  # This is NOT at mathematical (2,3)!

# ❌ Area calculation with wrong positioning
rect = Rectangle(width=1, height=2).shift(UP)  # Not tied to axes!
```

**CORRECT Examples**:
```python
# ✅ Point at mathematical (2,3)
point = Dot(axes.c2p(2, 3))

# ✅ Rectangle from x=1 to x=2, height f(1.5)
x_left, x_right = 1, 2
height = f(1.5)
rect_width = axes.x_axis.unit_size * (x_right - x_left)
rect_height = axes.y_axis.unit_size * height
rect = Rectangle(width=rect_width, height=rect_height)
rect.move_to(axes.c2p((x_left + x_right)/2, height/2))
```

**Geometric Object Creation Patterns**:
```python
# Circle with radius r centered at (h, k)
circle = Circle(radius=r * axes.x_axis.unit_size).move_to(axes.c2p(h, k))

# Line from (x1,y1) to (x2,y2)
line = Line(start=axes.c2p(x1, y1), end=axes.c2p(x2, y2))

# Tangent line at x=a
slope = derivative_value  # Calculate f'(a)
tangent = axes.plot(
    lambda x: f(a) + slope * (x - a),
    x_range=[a-1, a+1],
    color=GREEN
)

# Point on a curve
x_val = 2
point = Dot(axes.c2p(x_val, f(x_val)), color=RED)
```

=== COMMON GEOMETRIC ERRORS TO AVOID ===

**Error 1: Mixing Coordinate Systems**
```python
# ❌ WRONG
curve = axes.plot(lambda x: x**2)
point = Dot(UP * 2)  # Not on the curve!

# ✅ CORRECT
curve = axes.plot(lambda x: x**2)
x_val = 2
point = Dot(axes.c2p(x_val, x_val**2))  # Actually on curve
```

**Error 2: Ignoring Axis Scale**
```python
# ❌ WRONG - Rectangle doesn't match axis units
rect = Rectangle(width=1, height=2)

# ✅ CORRECT - Rectangle sized to axis units
dx = 0.5
height = f(x)
rect = Rectangle(
    width=axes.x_axis.unit_size * dx,
    height=axes.y_axis.unit_size * height
)
```

**Error 3: Incorrect Area Shading**
```python
# ❌ WRONG - Manual polygon might be misaligned
area = Polygon(point1, point2, point3, point4)

# ✅ CORRECT - Use built-in method
area = axes.get_area(curve, x_range=[a, b], color=BLUE, opacity=0.5)
```

**Error 4: Wrong Tangent Line**
```python
# ❌ WRONG - Just drawing any line
tangent = Line(start, end)

# ✅ CORRECT - Using actual derivative
x0 = 2
slope = 2 * x0  # derivative of x^2 at x=x0
y0 = x0**2
tangent = axes.plot(
    lambda x: y0 + slope * (x - x0),
    x_range=[x0-1, x0+1],
    color=GREEN
)
```
