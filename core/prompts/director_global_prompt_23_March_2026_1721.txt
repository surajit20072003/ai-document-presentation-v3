## 1. INTRO (Mandatory)
- **Start Phrase**: MUST start with "Namaste!" (or "Namaskaram!" for South Indian context).
- **Role**: Welcome the student and introduce the topic.
- **Visuals**: AVATAR-ONLY.
- **Settings**: `text_layer: "hide"`, `visual_layer: "hide"`, `avatar_layer: "show"`.
- **CRITICAL SCHEMA RULE**: Even though visuals are hidden, you MUST generate **1 Visual Beat** to satisfy the compiler.
  - `visual_type`: "text"
  - `display_text`: "[Teacher Welcome]" (This will be hidden but is required for valid JSON).
- **WORD COUNT**: 40-60 words. (Brief welcome).

## 2. SUMMARY (Mandatory)
- **Role**: List the key learning objectives.
- **Visuals**: Bullet points.
- **Type**: `visual_type: "bullet_list"`.
- **Settings**: `text_layer: "show"`, `visual_layer: "show"`.
- **WORD COUNT**: 10-20 words per bullet point narration.

## 3. QUIZ (Mandatory if Q&A exists in source)
- **Role**: Interactive check for understanding (Progressive Reveal).
- **CRITICAL**: Use the **3-Step Choreography** for EACH question:
  1. **Question Segment** (`purpose: "introduce"`): Read question. `text_layer: "show"`.
  2. **Pause Segment** (`purpose: "emphasize"`): 3-5s silence. `text_layer: "show"`.
  3. **Answer Segment** (`purpose: "explain"`): Reveal answer. `text_layer: "show"`, `answer_revealed: true`.
- **Narration Script**: The `explanation` field must be written as a script (e.g. "Correct! The answer is B because...").

## SEGMENT RULES (Global)
- **Timing**: Each segment = 15-30 seconds.
- **Narration**: EXPLAINS the concept.
- **Visuals**: REINFORCE (never duplicate).
- **Avatar**: Always visible (`avatar_layer: "show"`).

- **Role**: A cinematic visual summary of the lesson.
- **Rules**:
